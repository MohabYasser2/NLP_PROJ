قَالَ أَبُو زَيْدٍ أَهْلُ تِهَامَةَ يُؤَنِّثُونَ الْعَضُدَ ــ وَبَنُو تَمِيمٍ يُذَكِّرُونَ ،،،
وَالْجَمْعُ أَعْضُدٌ وَأَعْضَادٌ مِثْلُ أَفْلُسٍ وَأَقْفَالٍ ( 24 / 355 )

بِ مَنْزِلَةِ أَهْلِ الذِّمَّةِ إذَا دَخَلُوا قَرْيَةً مِنْ قُرَى أَهْلِ الْحَرْبِ
ثُمَّ ظَفِرَ الْمُسْلِمُونَ بِ هَا فَهُمْ فَيْءٌ أَجْمَعُونَ إلَّا مِنْ عُرِفَ أَنَّهُ ذِمِّيٌّ .

الْمَسْأَلَةُ الْخَامِسَةُ قَوْلُهُ تَعَالَى : { الَّذِينَ يُظَاهِرُونَ }
حَقِيقَتُهُ تَشْبِيهُ ظَهْرٍ [ بِ ظَهْرٍ ] وَالْمُوجِبُ لِلْحُكْمِ

قالَ عَلِيٌّ : «إِنَّ الْعِلْمَ نُورٌ، وَالْجَهْلَ ظُلْمَةٌ»
لكن… هل تَعْرِفُ ما معنى ٢٠٢٥؟

إِلَىٰ   السَّمَاءِ… ذَهَبَ فَتًىً   يَسْعَىٰ
بِ قَلْبٍ صَافٍۭۭۭ ، وَقَالَ : أَأَنْتَ مُؤْمِنٌ بِ لْحَقِّ؟

وَقِيلَ : غُسْلُ الْمَيِّتِ آكَدُ مُطْلَقًا !!!
( وَسُئِلَ ) رَحِمَهُ اللَّهُ عَنْ قَوْلِهِمْ إنَّ أَطْفَالَ الْكُفَّارِ فِي الْجَنَّةِ

وَلَمْ يُعْرَفْ فِي شَيْءٍ مِنْ الْآثَارِ لَفْظٌ وَاحِدٌ
مِمَّنْ بَاشَرَهَا مِنْ الصَّحَابَةِ رضي الله عنهم

7318 - ذُكِرَ عَنْ أَبِي بَدْرٍ شُجَاعِ بْنِ الْوَلِيدِ، عَنْ أَبِي سِنَانٍ الشَّيْبَانِيِّ
قال : فَتَحَ عَلَيْهِمْ أَرْبَعِينَ سَنَةً ( ؟؟ )

[ قَوْلُهُ : وَهُوَ الثَّمَنُ وَرِبْحُهُ ]
أَيْ وَحِصَّةُ رَبِّهِ مِنْ رِبْحِهِ ( 15 / 296 )

تَجْرِي بِأَعْيُنِنَا جَزَاءً لِمَنْ كَانَ كُفِرَ
وَإِنْ قَالَ : مَتَى مَرِضْتُ أَوْ عَرَضَ لَيَّ عَارِضٌ خَرَجْتُ

وَفِي صَحِيحِ مُسْلِمٍ عَنْ أَبِي هُرَيْرَةَ
قَالَ : قَالَ رَسُولُ اللَّهِ ﷺ : { مَنْ تَوَضَّأَ يَوْمَ الْجُمُعَةِ }

بِ لَا نِزَاعٍ ( وَإِنْ كَانَتْ مُجْبَرَةً : فَعَلَى الْوَلِيِّ )
وَتَقْرِيرُ الْأُصُولِ أَنَّ الْأَوَّلَ تَعَلَّقَ قَبْلَ الثَّانِي
